Welcome to the Sentiment Analysis Dashboard!

The primary goal of this system is to help you understand public sentiments related to fuel price increases and other topics more efficiently. You can identify whether the general sentiment is leaning towards positive, negative, or neutral regarding the chosen topic, providing valuable insights. We hope this application will assist you in analyzing and comprehending public sentiments related to fuel price increases and other topics more effectively. Enjoy using the Sentiment Analysis Dashboard!

Application Workflow

  1. Upload Data: Upload a dataset containing text or reviews related to the topic you want to analyze sentiments for.
  2. Preprocessing: Perform data preprocessing, such as removing punctuation, converting text to lowercase, and more.
  3. Feature Extraction: Select relevant features from the text data, such as unigrams, bigrams, or TF-IDF.
  4. Model Training: Train the Naive Bayes Classifier based on the processed data and extracted features.
  5. Sentiment Analysis: Use the trained model to analyze sentiments in the input text.
